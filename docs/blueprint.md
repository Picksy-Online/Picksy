# **App Name**: Picksy

## Core Features:

- Product Browsing: Browse available products through the browse page and collection pages.
- Seller Profiles: View seller profiles and their products on the store pages.
- AI-Powered Product Recommendations: Suggest related products to users using AI, enhancing the shopping experience.
- User Authentication: Secure login and profile management for users.
- Admin Dashboard: Manage categories, disputes, and products through the admin dashboard.
- Sales Analytics: Display sales analytics and data for sellers and admins in a dedicated page.
- AI Content Moderation Tool: Implement a tool to automatically identify policy violations and fake or malicious products.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and reliability.
- Background color: Light blue (#E8EAF6), a very desaturated version of the primary color to create a clean backdrop.
- Accent color: Purple (#7E57C2), providing a contrasting hue for interactive elements.
- Body font: 'Inter', a sans-serif for modern readability.
- Headline font: 'Space Grotesk', a geometric sans-serif for impactful headlines.
- Use a set of minimalist icons to represent product categories, user actions, and settings.
- Employ a grid-based layout system for consistent spacing and alignment of content.
